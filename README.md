# Quicker Deliver App

Simple web-based delivery service application.  
Users can create orders, track delivery location, and calculate delivery cost based on distance.

## Features
- Create and manage delivery orders
- Real-time location display (map integration)
- Distance-based price calculation
- Simple and clean web interface

## Tech Stack
- Python / Django (Backend)
- HTML, CSS, JavaScript or React (Frontend)
- PostgreSQL (Database)
